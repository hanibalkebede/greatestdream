# Snake Highway

Open `index.html` in a browser to play.

Controls (keyboard):
- A / D : move left / right between lanes
- Space : start / pause
- W : increase speed
- S : decrease speed
- E : random color
- R : breathe fire
- G : breathe water
- Q : toggle ghost (invincible)
- T : transform to a random animal
- F : transform to car
- C : transform to fish
- Y : shrink size

Gameplay: choose a snake on the start screen, press space to start moving. Collect yellow points and purple big points to grow and score.
